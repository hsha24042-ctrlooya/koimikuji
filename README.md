おみくじアプリ（恋愛バージョン）

使い方
ボタンを押すと、恋愛運がランダムに表示されます。
恋みくじアプリ（最強）
ボタンをクリックするだけで、今日の恋愛運を占うことができるシンプルなWebアプリケーションです。ピンクを基調とした可愛らしいデザインで、ランダムに運勢が表示されます。

機能
おみくじ抽選機能: 「おみくじを引く」ボタンを押すと、4種類の運勢（大吉、中吉、小吉、凶）からランダムに1つが表示されます。

レスポンシブなデザイン: シンプルなスタイル設定により、ブラウザ上で快適に動作します。

ファイル構成
プロジェクトは以下の3つのファイルで構成されています。

Plaintext
.
├── index.html    # アプリケーションの構造（HTML）
├── style.css     # ページのデザイン・装飾（CSS）
└── script.js     # おみくじの抽選ロジック（JavaScript）
使用技術
HTML5: ページの骨組み、ボタン、結果表示エリアの作成

CSS3: 背景色、ボタンの装飾、レイアウト

JavaScript (ES6): 乱数生成による運勢の抽選とDOM操作

使い方
すべてのファイル（index.html, style.css, script.js）を同じフォルダ内に保存します。

ブラウザで index.html を開きます。

「おみくじを引く」ボタンをクリックすると、画面に結果が表示されます。

カスタマイズ方法
script.js 内の fortunes 配列の内容を書き換えることで、おみくじの結果（テキストや絵文字）を自由に変更・追加することが可能です。

JavaScript
const fortunes = [
    "大吉💛素敵な出会いがあるかも！",
    "中吉：少し勇気を出すとよいことが起きる",
    // ここに新しい運勢を追加できます
];
注意点
script.js 内のスペルミス（foetunes.lenght など）を修正すると正常に動作します。

style.css 内のホバーアクション（btton:hover）のスペルを button:hover に修正すると、ボタンにマウスを乗せた際の色変化が有効になります
